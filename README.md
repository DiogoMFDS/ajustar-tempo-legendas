# Ajustar Tempo de Legendas

Este projeto é uma ferramenta simples para ajustar os tempos de legendas em arquivos .srt (SubRip). Permite sincronizar legendas com vídeos, corrigindo atrasos ou adiantamentos.

## Funcionalidades
- Carregar arquivo de legendas .srt
- Ajustar tempos de início e fim de cada legenda
- Salvar o arquivo ajustado

## Instalação
1. Clone o repositório:
   ```
   git clone https://github.com/seu-usuario/ajustar-tempo-legendas.git
2. pip install -requirements
   ```

## Uso
1. coleque as legendas com o tempo para ajustar na pasta docs/leg-originais
2. execute o .bat e escolhe os ajustes de tempo
